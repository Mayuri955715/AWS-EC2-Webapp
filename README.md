# AWS EC2 Web Application

## Project Overview

This project deploys a static website on an AWS EC2 Ubuntu server using Nginx.

The deployment is automated using a Shell Script.

## Technologies Used

- AWS EC2
- Ubuntu
- Linux
- Nginx
- Shell Scripting
- Git
- GitHub

## Files

- index.html
- setup.sh
- README.md

## How to Run

```bash
chmod +x setup.sh
./setup.sh
```
